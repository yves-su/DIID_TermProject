# Android 手機連線設定教學

為了將 APP 安裝到您的手機進行測試，您需要啟用「開發者模式」與「USB 偵錯」。因為此 APP 需要藍牙功能，**無法使用電腦模擬器**，必須使用實體手機。

## 步驟 1: 啟用開發者選項 (Developer Options)

1.  開啟手基的 **「設定 (Settings)」**。
2.  滑到最下方，點選 **「關於手機 (About Phone)」**。
3.  找到 **「版本號碼 (Build Number)」** (通常在「軟體資訊」裡面)。
4.  **連續點擊版本號碼 7 次**，直到畫面出現「您現在已成為開發人員！」的提示。

## 步驟 2: 開啟 USB 偵錯 (USB Debugging)

1.  回到 **「設定」** 首頁。
2.  進入 **「系統 (System)」** 或 **「更多設定」**，找到新出現的 **「開發者選項 (Developer Options)」**。
3.  進入後，找到 **「USB 偵錯 (USB Debugging)」** 並將其**開啟**。

## 步驟 3: 連接電腦

1.  使用 USB 傳輸線將手機連接到電腦。
2.  **注意手機螢幕**：會跳出一個視窗詢問「是否允許 USB 偵錯？(Allow USB debugging?)」。
3.  勾選「一律允許透過這台電腦進行 (Always allow...)」，然後點選 **「允許 (Allow)」**。

## 步驟 4: 驗證連線

回到 VS Code 的終端機，輸入以下指令：

```bash
flutter devices
```

如果您看到類似以下的輸出，代表成功了：

```
Pixel 7 (mobile) • 12345678 • android-arm64 • Android 14 (API 34)
```

## 常見問題

*   **Q: 插上線了但沒反應？**
    *   A: 請確認您的 USB 線是可以傳輸資料的（有些線只能充電）。
*   **Q: 找不到開發者選項？**
    *   A: 不同品牌的手機位置可能不同，請Google搜尋「您的手機型號 + 開發者模式」。
